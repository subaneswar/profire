const int flamePin = A0;
const int ledPin = 6;
const int buzzerPin = 3;

In this Code we will setting the pin of buzzer,sensor and led

flamePin(SENSOR) analog pin A0
LED Digital pin 6
BUZZER Digital pin 3

void setup() {

pinMode(flamePin, INPUT);

pinMode(ledPin, OUTPUT);

pinMode(buzzerPin, OUTPUT);

Serial.begin(9600); }

  1. Three constant integer variables: flamePin, ledPin, and buzzerPin are defined.
  2. These variables hold the pin numbers for connected components.
  3. Adjust these pin numbers to match your hardware configuration.

  5. In the setup() function:
    • pinMode(flamePin, INPUT);: Sets flamePin as an input for the flame sensor's analog output.
    • pinMode(ledPin, OUTPUT);: Defines ledPin as an output for an LED.
    • pinMode(buzzerPin, OUTPUT);: Specifies buzzerPin as an output for a buzzer.
    • Serial.begin(9600);: Initializes serial communication at 9600 baud for data exchange with a connected computer.
  1. Variable Definitions:
    • flamePin, ledPin, and buzzerPin are defined to store component connection pin numbers.
  2. Setup Function:
    • setup() initializes:
    • flamePin as input for the flame sensor.
    • ledPin as output for the LED.
    • buzzerPin as output for the buzzer.
    • Serial communication at 9600 baud rate.
  3. Loop Function:
    • loop() repeatedly:
    • Reads flame intensity from the sensor.
    • If intensity is above 250:
    • Turns on the LED and activates the buzzer.
    • Sends "Fire detected!" to serial.
    • If intensity is below 250:
    • Turns off the LED and deactivates the buzzer.
    • Waits for 1 second before repeating.
